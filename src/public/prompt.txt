# Números Únicos

## Descrição
Crie uma função que receba um array de números inteiros e retorne a soma dos números que aparecem **apenas uma vez** no array.  
Se um número aparecer mais de uma vez, ele **não deve ser incluído na soma**.

## Entrada
Um **array de números inteiros**.

## Saída
Um **número inteiro** representando a soma dos números que aparecem apenas uma vez no array.

## Regras / Restrições
- O array pode conter números **positivos e negativos**  
- O array pode conter **números repetidos**  
- Se **nenhum número for único**, a função deve retornar **0**  
- O tamanho do array pode variar de **1 até 1000 elementos**

## Exemplos

sumUnique([1, 2, 3, 2]) -> 4  
Explicação: apenas **1** e **3** aparecem uma única vez

sumUnique([4, 5, 4, 6]) -> 11  
Explicação: apenas **5** e **6** aparecem uma única vez

sumUnique([7, 7, 7]) -> 0  
Explicação: nenhum número aparece apenas uma vez
